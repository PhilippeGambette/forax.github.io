Dictionnaire / Objet

Dictionary ?

“In computer science, an associative array, map, symbol table, or dictionary is an abstract data type composed of a collection of (key, value) pairs, such that each possible key appears just once in the collection.”

source: Wikipedia.com - Dictionary

Dictionnaire ?

“En informatique, un tableau associatif, map, table des symboles, ou dictionnaire est un type abstrait composé d'une collection de pairs (clé, valeur), tel que chaque clé possibe apparait une seul fois dans la collection.”

source: Wikipedia.com - Dictionnaire

Dictionnaire

Un dictionnaire en Javascript se déclare entre { et }.
Une clé et une valeur sont séparés par un :, les couples sont séparés par des virgules.

 

 

En Javascript, les clés sont forcéments des strings !

Accéder aux valeurs

L'opération '.' (dot) permet d'obtenir la valeur correspondant à une clé.

 

 

Dans la cas ou la clé n'existe pas, la valeur est undefined

Accéder aux valeurs (2)

Comme pour les tableaux, il est possible d'utiliser les [ ] pour obtenir la valeur correspondant à une clé.

 

 

on utilise . si la clé est constante, [] si la clé est une string

Dictionnaire dynamique

Il est possible de changer la valeur pour une clé ou d'ajouter une paire clé/valeur.

 

 

Pas de clé en double !

Le dictionnaire index les valeurs par leur clé, il ne peut y avoir qu'une seule valeur pour une clé.

 

 

Object vs Dictionary

En JavaScript, les dictionnaires ne sont pas appelés 'dictionary' mais 'object'

L'affichage par défaut (toString()) est pas intéressant, on utilise JSON.stringify() à la place

 

 

JSON = JavaScript Object Notation

Le tableau des clés

La fonction Object.keys permet d'avoir un tableau des clés d'un dictionnaire

 

 

Parcourir un dictionnaire

On peut pas parcourir un dictionnaire directement, il faut parcourir ses clés, puis pour chaque clé on peut aller chercher la valeur.

 

Parcourir un dictionnaire (2)

Il existe une syntaxe spéciale pour parcourir les dictionnaires, for(variable in dict)

 

Cette syntaxe est éuivalente au code qui utilise Object.keys()

Regrouper des propriétés

Un dictionnaire permet de regrouper des propriétés ayant attrait à un même concept ce qui permet de structurer le code
